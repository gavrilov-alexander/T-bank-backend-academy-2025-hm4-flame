## Описание

Этот проект состоит из нескольких скриптов, которые выполняют следующие задачи:

1. **Тестирование базовой функциональности**: проверка успешного выполнения программы и создания изображения.
2. **Тестирование свойств изображения**: проверка, что сгенерированное изображение является корректным PNG-файлом с содержимым.
3. **Тестирование производительности многопоточности**: сравнение времени выполнения программы с разным количеством потоков.

---

## Скрипты

### `test_basic_functionality.sh`

Этот скрипт проверяет базовую функциональность генератора фрактальных изображений. Он запускает программу с заданными параметрами и проверяет:

1. Успешное выполнение программы (код возврата 0).
2. Создание выходного файла изображения.

### `test_image_properties.sh`
Этот скрипт проверяет свойства сгенерированного изображения. Он выполняет следующие проверки:

1. Существует ли файл изображения.
2. Имеет ли файл расширение .png.
3. Имеет ли файл ненулевой размер.
4. Является ли файл корректным PNG-файлом (проверяется сигнатура PNG).
5. Если файл изображения отсутствует, скрипт автоматически генерирует его, используя программу.

### `test_multithreading_performance.sh`
Этот скрипт тестирует производительность программы при использовании разного количества потоков. Он измеряет время выполнения программы с 1, 2 и 4 потоками и сохраняет результаты в файл performance_results.csv.


## Использование
Для запуска всех тестов используйте скрипт run.sh. Он автоматически выполнит все три программы, передавая путь к dll файлу в качестве аргумента.

## Требования
1. Dotnet: Убедитесь, что DOTNET SDK установлена и доступна в командной строке.
2. Bash: Скрипты написаны на Bash и требуют установленного интерпретатора Bash.
3. Права доступа: Убедитесь, что у вас есть права на выполнение скриптов.
